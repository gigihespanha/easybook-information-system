# EasyBook – Systems Analysis & Database Design

EasyBook is a mobile appointment management information system designed for small businesses to manage bookings efficiently and prevent scheduling conflicts.

This project was independently developed as part of a Systems Analysis course, covering the full system lifecycle from requirements definition to database implementation.

## What’s included
- System description and business analysis
- Level 0, Level 1, and Level 2 DFDs
- Entity-Relationship (E-R) modeling
- Relational schema mapping
- Microsoft Access database implementation
- UI design for mobile users

## Files
- `/documentation/` → PDF + diagrams (DFD, ERD, relationship, UI)
- `/database/` → Microsoft Access database file (.accdb)

## Tools
- Microsoft Access
- Data modeling / ERD
- DFD process modeling
